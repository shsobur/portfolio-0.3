## 🚀 Key Features

- **Performance Balanced Animations:** Optimized using Framer Motion with GPU-acceleration to ensure zero lag on mobile browsers.
- **Glassmorphic UI:** A clean, modern aesthetic using Tailwind CSS and backdrop-blur effects.
- **Staggered Scroll Reveals:** Content reveals itself elegantly as the user scrolls, creating a premium "Apple-style" experience.
- **Dynamic Video Gallery:** High-impact work showcase with an integrated Vimeo player modal.
- **Responsive Contact System:** Fully functional contact form powered by **EmailJS** with instant success/error feedback.
- **Accessibility:** Respects `prefers-reduced-motion` settings for users with motion sensitivity.

## ⚙️ Local Development

Follow these steps to set up the project locally:

1. **Clone the repository:**

   ```bash
   git clone <your-repo-url>
   ```

   > ⚠️ Replace `<your-repo-url>` with the actual repository URL (private or public).

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your EmailJS credentials:

   ```env
   VITE_SERVICE_ID=...!
   VITE_TEMPLATE_ID=...!
   VITE_USER_ID=...!

   For env file contact with me.
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
